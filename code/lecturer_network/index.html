Diese Visualisierung zeigt das Netzwerk der Kursleiter*innen der Bauhaus-Universität Weimar.
Je mehr Veranstaltungen von einer Person geleitet werden umso größer ist ihr Knoten im Netzwerk. Falls eine Veranstaltung von mehreren Personen geleitet wird sind diese Personen im Netzwerk mit Kanten verbunden. Je näher zwei Personen beieinander sind, umso mehr Veranstaltungen leiten sie gemeinsam.

Kurze Tipps zur Interpretation des Diagramms:

  • In das Diagramm lässt sich durch Drehen des Mausrads oder einem Fingerpinch reinzoomen. Und die aktuelle Ansicht kann mit gedrückt gehaltener Maus verschoben werden.
  • Beim Hovern über eine Person, werden alle Personen farblich markiert die mit dieser Personen verbunden sind.
  • Ein Klick auf eine Person leitet auf eine Auflistung aller Veranstaltungen, die von der Person geleitet wird, weiter.
  • Ein Klick auf eine Verbindung zwischen zwei Personen leitet auf eine Auflistung aller Veranstaltungen, die von beiden Personen geleitet werden, weiter.
