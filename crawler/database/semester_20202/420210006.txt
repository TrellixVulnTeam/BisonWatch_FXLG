# Beschreibung
Beschreibung :3 S: Personal web pages are the figurehead for academics worldwide to showcase their work to the scientific community at large. In this project we explore a large collection of academic web pages collected from web archives and university department sites. Using the contemporary data analysis toolbox, we will investigate questions concerning language use in such pages and how it changes over time, how bibliometric and scientometric indicators correlate with attributes of self-presentation, as well as issues of publication ethics and how they might be studied in the context of self-presentation.
