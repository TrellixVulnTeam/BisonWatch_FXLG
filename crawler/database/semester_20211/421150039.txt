# Beschreibung
Beschreibung :3 S: In this 3-day hands-on workshop we will explore the design and basic implementation of musical interfaces and sound generation with PSLab, Processing 3 and Python. , <br/>, The Pocket Science Lab 1 is an open hardware device, which can be used as oscilloscope, multimeter, wave generator, logic analyzer, power source, among others. In this class we will use the PSlab as an input and output device, to generate creative/procedural algorithmic music. , <br/>, At least since Brian Enos Generative Music 1 (1996), the idea of ever changing musical compositions created by a system has been popularized. By connecting sensors to the PSLab we have the foundation for creating our own physical/tangible musical interfaces. , <br/>, For example ultrasonic sensors can be used similar to a theremin, capacitive sensing can trigger touch events, and a light dependend resistor (LDR) can be controlled via a flash light., <br/>, To do so we will shed some light on basic electronics and interfacing methods, sound syntheis and Programming in Processing 3 2 (and some Python).
